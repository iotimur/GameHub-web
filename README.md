### Онлайн-магазин компьютерных игр

---

**Описание проекта:**

GameHub-web — это онлайн-магазин, созданный для любителей компьютерных игр. Мы предлагаем широкий ассортимент игр, включая новинки и бестселлеры, а также удобный интерфейс для покупок и управления личным кабинетом. Наша цель — предоставить пользователям лучший опыт в выборе и покупке игр.

---

# MVP1: GameHub-web

- **Главная страница (Мухаметов Тимур)**
  - Логотип магазина
  - Главное навигационное меню, кнопки для входа и регистрации пользователей
  - Новые продукты и бестселлеры

- **Корзина покупок (Майорова Анастасия)**
  - Список товаров
  - Ценовой лист
  - Описание игр

- **Страница игры (Макарова Екатерина)**
  - Информация о продукте
  - Системные требования
  - Комментарии и рейтинги
  - Цена

- **Категории (Ахмадуллина Эмилия)**
  - Сортировка игр
  - Топ-продаж
  - Новинки
  - Специальные предложения

- **Личный кабинет (Федорова Елена)**
  - Вход или авторизация
  - Личная информация
  - Мои покупки
  - Помощь

## MVP2: Расширенные функции GameHub-web

### Новые возможности для пользователей:

- **Поиск по играм (Мухаметов Тимур)**
  - Удобный и интуитивно понятный поиск по всей базе игр, позволяющий пользователям легко находить интересующие их продукты.
  - Возможность добавления игр в корзину прямо из результатов поиска.
  - Отдельная страница с выводом списка игр, организованным по категориям, где пользователи могут быстро находить нужные им жанры и новинки.

- **Промокоды и скидки (Майорова Анастасия)**
  - Внедрение функции добавления промокодов, позволяющей пользователям получать скидки на покупки.
  - Создание базы данных для хранения промокодов, что обеспечит гибкость в предложениях и акциях.

- **Улучшенная система комментариев (Макарова Екатерина)**
  - Возможность сортировки комментариев по различным критериям, чтобы пользователи могли быстро находить самые полезные отзывы.
  - Функция лайков для комментариев, что позволит выделять наиболее ценные мнения.
  - Добавление поддержки английского языка, если время позволит, чтобы расширить аудиторию.
  - Интерактивная галерея изображений игр для более глубокого погружения в мир игровых новинок.

- **Фильтрация и избранное (Ахмадуллина Эмилия)**
  - Функция фильтрации категорий, позволяющая пользователям быстро находить игры по интересующим их критериям.
  - Возможность добавления игр в избранное, чтобы сохранить самые желаемые продукты для будущих покупок.
  - Отдельная страница избранного для удобного доступа к сохраненным играм.

- **Обновленный интерфейс (Федорова Елена)**
  - Изменение футера и улучшение функционала входа и регистрации, чтобы сделать процесс более удобным и приятным.
  - Создание кнопки для быстрого перехода из личного кабинета в раздел избранного, упрощая навигацию по сайту.
